## Deploy to Vercel in 3 steps

### Step 1 — Upload to GitHub
1. Go to github.com → New Repository → name it `sparkmidwest`
2. Upload all files from this folder into the repo
3. Commit

### Step 2 — Deploy on Vercel
1. Go to vercel.com → Add New Project
2. Import `sparkmidwest` from GitHub
3. Framework auto-detects as **Vite**
4. Click **Deploy** — live in ~2 minutes

### Step 3 — Connect sparkmidwest.com
1. Vercel → your project → Settings → Domains
2. Add: `sparkmidwest.com` and `www.sparkmidwest.com`
3. Copy the DNS records Vercel gives you
4. Add them to your domain registrar (Namecheap/GoDaddy)
5. Live in 10-30 minutes
